# Milestone 1

## Goal

Bootstrap DisturbMyLive with ACP and a minimal iOS app scaffold.

## Scope boundary

Create repo structure, ACP artifacts, project scaffold, and basic verification targets.

## Deliverables

- XcodeGen project file
- SwiftUI app scaffold
- ACP design files
- ACP task files
- progress.yaml
- local index

## Success criteria

- Repo structure exists
- ACP structure exists
- progress.yaml points to next executable task
- app can be generated into an Xcode project

## Required verification surface

- file presence
- xcodegen generation

## Blockers

- missing xcodegen
- missing Xcode command line tools

## Exit condition

Bootstrap files exist and next milestone is ready for execution.

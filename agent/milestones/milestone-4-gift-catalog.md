# Milestone 4

## Goal

Build the gift and interaction catalog surface.

## Scope boundary

Present gift icons and descriptions of the sound played for each gift or interaction.

## Deliverables

- catalog models
- catalog UI
- icon loading path
- sound mapping data shape

## Success criteria

- list is visible after connect
- gift and interaction entries are distinct
- icon and description shape is implementation ready

## Required verification surface

- build
- tests
- manual simulator proof

## Blockers

- official gift metadata source
- final sound design decisions

## Exit condition

Catalog surface is ready for real data and playback integration.
